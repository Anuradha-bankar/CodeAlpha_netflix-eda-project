# netflix-eda-project
# Exploratory Data Analysis (EDA)

## Description
This project performs Exploratory Data Analysis (EDA) on Netflix dataset to find insights.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Tasks Performed
- Data Cleaning
- Handling Missing Values
- Removing Duplicates
- Data Visualization

## Dataset
Netflix Movies and TV Shows dataset

## Output
Cleaned dataset and visualizations

## Insights
1.Netflix has significantly more Movies compared to TV Shows.
2.The number of shows and movies added to Netflix increased rapidly after 2015.
3.The majority of Netflix content is produced by a few countries.
4.TV-MA is the most common rating on Netflix.
5.Drama, International Movies, and Comedies are among the most popular genres.
6.Most movies on Netflix have a duration between 80 to 120 minutes.

## Overall Conclusion:
Netflix content is dominated by Movies, with rapid growth in recent years.
The platform focuses heavily on mature content and popular genres like Drama and Comedy.
A few countries contribute the majority of content, and movie durations follow standard industry patterns.

## Hypothesis Testing:
| Hypothesis | Statement | Result |
|-----------|----------|--------|
| H1 | Movies are more than TV Shows | Accepted |
| H2 | Content increased after 2015 | Accepted |
| H3 | TV-MA is a common rating | Accepted |
